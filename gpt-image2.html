完整操作步驟

三步驟一鍵出圖

1

請 Claude 幫你生提示詞

不要直接在 GPT 裡亂下 prompt——先告訴 Claude 你的品牌風格、想要什麼版面、要放什麼文字，讓 Claude 幫你生出精準的提示詞。

// 貼給 Claude 的範本

我要做一張 Instagram 限時動態圖片。
品牌：飲食陪跑顧問，溫暖專業風格
素材：我的生活照＋產品照（我會上傳）
版面：人物在右側，文字在左上，產品在左下
主標：吃對了，身體自然瘦
副標：不是節食，是讓身體找回平衡
底部：@你的帳號名稱

請幫我生成一個完整的 GPT Image-2 提示詞。
2

把提示詞貼到 ChatGPT，上傳照片

確認 ChatGPT 用的是 Image-2 模型（不是 DALL-E），把 Claude 生的提示詞貼進去，同時上傳你的產品照和生活照。

// 注意事項

✓ 確認模型選的是 Image-2（不是 DALL-E 3）
✓ 照片上傳 2–3 張即可，不要太多
✓ 提示詞直接貼，不要修改
✓ 送出後等待約 30–60 秒
3

確認中文字＋直接存圖發布

圖生出來後，檢查中文字有沒有跑掉。沒問題直接存圖，上傳 IG 限時動態或貼文。

// 想微調？直接在同一個對話說

「把主標題放大」
「產品移到右下角」
「背景換成白色」
「整體色調調暖一點」

不用重新生，直接修改就好
進階應用

輪播貼文
也可以一次生好幾張

這是 Image-2 最強的地方——以前全擠一張大圖要自己裁，現在每張獨立生成、風格一致。

Step 1｜請 Claude 規劃輪播結構

告訴 Claude 你這則輪播的主題、要幾張、每張要講什麼，讓它幫你規劃好每張的版面和文字。

我要做一則 IG 輪播貼文，共 5 張。
主題：為什麼節食反而瘦不下來
風格：溫暖手繪插圖，白色背景
每張：一個觀點＋簡短說明＋情境插圖

請幫我規劃 5 張的結構，
並為每張生成 GPT Image-2 的提示詞。

Step 2｜在 ChatGPT 一張一張生成

把 Claude 給的每張提示詞，一個一個貼到 ChatGPT Image-2，分開生成，每張都是獨立的成品圖。

// 重要：每張分開送出

✓ 第 1 張提示詞 → 生成 → 存圖
✓ 第 2 張提示詞 → 生成 → 存圖
✓ 第 3 張提示詞 → 生成 → 存圖
✓ ...

風格一致的關鍵：
在每張提示詞都加上相同的風格描述，
例如「溫暖手繪風格，白色背景，同系列」

Step 3｜直接上傳 IG 輪播

5 張圖全部存好，直接在 IG 選「新增貼文 → 選取多張」，按順序上傳，輪播貼文完成。

// 發布前確認清單

✓ 每張中文字正確
✓ 風格視覺一致
✓ 順序正確（封面圖放第一張）
✓ 文案貼上貼文說明欄
✓ 加上相關 hashtag
